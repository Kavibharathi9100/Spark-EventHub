A complete **Event Management Web Application** built with Python and Flask, designed for colleges, clubs, and organizations to manage events efficiently.

---

## ✨ Features

- **Event Management**: Create, edit, and delete events with rich details
- **Smart Registration**: Gmail validation, free & paid events
- **UPI Payment Integration**: Dynamic QR code payment system
- **Date-Locked Attendance**: Mark attendance only on event day
- **Auto Certificate Generation**: Professional PDF certificates with event-specific themes
- **Email Automation**: Registration, payment, reminders & certificate emails
- **Admin Dashboard**: Full analytics and control panel
- **Certificate Verification**: Public verification system
- **My Tickets Portal**: Users can view all their registrations

---

## 🛠 Tech Stack

- **Backend**: Python + Flask
- **Database**: SQLite3
- **PDF Generation**: ReportLab
- **QR Code**: qrcode
- **Email**: Gmail SMTP (App Password)
- **Frontend**: HTML, CSS, Jinja2
